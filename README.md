# Epic Simcar Remix for EmulationStation Desktop Edition v2.0

A Remix between Epic Noir (Revisited) and SimCar Carousel theme set for ES-DE.

![systems](system.png)
![games](game.png)

The following options are included:

2 variants:

- With videos
- Without videos

Dark color scheme

4 aspect ratios:

- 16:9
- 16:10
- 4:3
- 21:9
